/*【*】程序简介

- 工程名称        ：MPU6050 姿态传感器基础实验  
- 实验平台        ：STM32F103ZET6开发板  

【 ！】功能简介：

本实验实现 MPU6050 姿态传感器的初始化，通过 I2C 读取加速度和陀螺仪原始数据，并通过串口周期性打印。

【 ！】实验操作：

    1. 编译并下载程序到开发板；
    2. 打开串口调试助手，观察 0.5s 周期输出的加速度与陀螺仪数据；
    3. 旋转传感器方向，可观察 X/Y/Z 轴加速度与角速度的变化；
*/
/*********************************************************************************************/

/*【*】 引脚分配



IIC_SCL -> PB6            PB6 作为硬件 I2C 的 SCL，引脚连接 MPU6050 的 SCL
IIC_SDA -> PB7            PB7 作为硬件 I2C 的 SDA，引脚连接 MPU6050 的 SDA

INT    -> PF7             MPU6050 的中断引脚（可选连接），用于数据就绪中断
AD0    -> GND             MPU6050 的地址选择引脚，默认接地（地址为0x68），若接高电平则地址为0x69

*/
/*********************************************************************************************/

/*【*】程序描述：

< main.c >

    1. 初始化 USART1，用于串口打印调试；
    2. 初始化 I2C2，并初始化 MPU6050；
    3. 读取加速度与陀螺仪的原始数据（X/Y/Z 三轴）；
    4. 主循环中每隔 0.5 秒读取一次数据并通过串口打印；
    5. 可在后续实验中加入姿态解算（如互补滤波或卡尔曼滤波）；
*/
/*********************************************************************************************/
